## Getting Started

### Prerequisites

- npm
  ```sh
  npm install --global yarn
  ```

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/AtulRaj151/revamp.git
   ```
2. Install NPM packages
   ```sh
   yarn install
   ```
3. Start locally
   ```sh
   yarn start
   ```
